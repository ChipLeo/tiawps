Tiawps documentation

Tiawps is a tool for decrypting sniffed wow sessions and storing them in sqlite databases. Sniffing and decrypting are separated processes.

Guide

  1. Get a sniffer which is able to store in the libpcap format. Tcpdump and Wireshark are known to do this.
  2. Start sniffing everything on TCP port 3724.
  3. Start wow and log in.
  4. Use tiawps_reader.bat to extract the encryption key from the running wow process. A KEY*.txt file will be created.
  5. Play around and do whatever you want to
  6. Stop wow and stop sniffing
  7. Save the sniffer dump in a libpcap compatible format
  8. Drag both KEY*.txt and *.cap to tiawps_decrypter.bat
  9. You should now see a window which says "Finished decrypting 1 of 1 connections". A *.sqlite should have been created which contains the decrypted session.
